# Hyperparameters

We provide details for every adjustable hyperparameter in this project.

This list might look long, but in practice very few hyperparameters need to be tuned, we just wanted to be thorough :)

Important hyperparameters:


| Hyperparameter            | Description                                                                                                                                          | Recommended/Legal Values                                                                                 |
|---------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| `batch_size`              | Number of samples in each mini-batch                                                                                                                 | Integer value, e.g., 8                                                                                   |
| `lr`                      | Learning rate                                                                                                                                        | Floating-point value, e.g., 0.00008                                                                      |
| `warmup_epoch`            | Number of warm-up epochs where only the posterior (enc-dec) is trained                                                                               | Integer value, default: 1                                                                                |
| `patch_size`              | Patch size for prior keypoint proposals / glimpses                                                                                                   | Integer value, e.g., 16                                                                                  |
| `n_kp_enc`                | Number of posterior keypoints to be learned                                                                                                          | Integer value, e.g., 45                                                                                  |
| `n_kp_prior`              | Number of prior keypoints / patches                                                                                                                  | Integer value, e.g., 64                                                                                  |
| `anchor_s`                | Glimpse size as a ratio of image size (posterior patch size)                                                                                         | Floating-point value, e.g., 0.25 for image_size 128                                                      |
| `scale_std`               | Prior standard deviation for object scales                                                                                                           | Floating-point value, typical range [0.15, 1.0] (0.15 for `anchor_s`=0.125, and 0.3 for `anchor_s`=0.25) |
| `offset_std`              | Prior standard deviation for object offsets                                                                                                          | Floating-point value, typical range [0.1, 1.0] (0.1 for `anchor_s`=0.125, and 0.2 for `anchor_s`=0.25)   |
| `recon_loss_type`         | Type of reconstruction loss                                                                                                                          | "vgg", "mse"                                                                                             |
| `beta_kl`                 | Weight for KL divergence loss on latents                                                                                                             | Floating-point value, e.g., 0.08                                                                         |
| `beta_obj`                | Weight for the transparency regularization loss                                                                                                      | Floating-point value, e.g., 0.08                                                                         |
| `image_size`              | Size (height/width) of the input image                                                                                                               | Integer value, e.g., 128                                                                                 |
| `n_views`                 | Number of input camera views                                                                                                                         | Integer value, typically 1                                                                               |
| `learned_feature_dim`     | Dimension of latent visual features extracted from foreground glimpses                                                                               | Integer value, e.g., 4                                                                                   |
| `learned_bg_feature_dim`  | Dimension of latent visual features extracted from the background                                                                                    | Integer value, e.g., 4                                                                                   |
| `context_dim`             | Dimension of context latent vector (latent actions)                                                                                                  | Integer value, e.g., 7                                                                                   |
| `action_condition`        | Flag to condition the model on action inputs                                                                                                         | `true`, `false`                                                                                          |
| `action_dim`              | Dimension of action vector (if action_condition is `true`)                                                                                           | Integer value, e.g., 7                                                                                   |
| `image_goal_condition`    | Flag to use image conditioning                                                                                                                       | `true`, `false`                                                                                          |
| `language_condition`      | Flag to use language conditioning                                                                                                                    | `true`, `false`                                                                                          |
| `language_embed_dim`      | Language embedding dimension (e.g. T5-large)                                                                                                         | Integer value, e.g., 1024                                                                                |
| `language_max_len`        | Maximum number of language tokens to allow                                                                                                           | Integer value, e.g., 32                                                                                  |
| `timestep_horizon`        | Number of timesteps for training the dynamics model                                                                                                  | Integer value, e.g., 20                                                                                  |
| `beta_dyn`                | Weight for KL dynamics loss                                                                                                                          | Floating-point value, e.g., 0.2                                                                          |
| `pint_ctx_layers`         | Number of transformer layers used for context modeling                                                                                               | Integer value, e.g., 4                                                                                   |
| `pint_ctx_heads`          | Number of attention heads in context transformer                                                                                                     | Integer value, e.g., 8                                                                                   |
| `pint_dyn_layers`         | Number of transformer layers in dynamics module (PINT dynamics)                                                                                      | Integer value, e.g., 6                                                                                   |
| `pint_dyn_heads`          | Number of attention heads in dynamics module                                                                                                         | Integer value, e.g., 8                                                                                   |
| `pint_dim`                | Hidden dimension of the PINT transformer modules                                                                                                     | Integer value, e.g., 512                                                                                 |


Full list:

| Hyperparameter            | Description                                                                                                                                          | Recommended/Legal Values                                                                                 |
|---------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| `ds`                      | Dataset name                                                                                                                                         | "sketchy", "obj3d128", "langtable", "phyre", etc.                                                        |
| `root`                    | Root directory for dataset                                                                                                                           | Example: /home/ubuntu/data/sketchy/data                                                                  |
| `device`                  | Device to run the model on                                                                                                                           | "cuda:0", "cuda", "cpu"                                                                                  |
| `batch_size`              | Number of samples in each mini-batch                                                                                                                 | Integer value, e.g., 8                                                                                   |
| `lr`                      | Learning rate                                                                                                                                        | Floating-point value, e.g., 0.00008                                                                      |
| `use_scheduler`           | Flag to enable learning rate scheduler                                                                                                               | `true`, `false` (we used `false`)                                                                        |
| `scheduler_gamma`         | Learning rate scheduler gamma parameter                                                                                                              | Floating-point value, e.g., 0.99                                                                         |
| `adam_betas`              | Beta values for the Adam optimizer                                                                                                                   | List of floats, default: (0.9, 0.999)                                                                    |
| `adam_eps`                | Epsilon value for the Adam optimizer                                                                                                                 | Floating-point value, e.g., 1e-6                                                                         |
| `weight_decay`            | Weight decay for the optimizer                                                                                                                       | Floating-point value, default: 0.0                                                                       |
| `load_model`              | Flag to load pre-trained model                                                                                                                       | `true`, `false`                                                                                          |
| `pretrained_path`         | Path to the pre-trained model                                                                                                                        | String or null                                                                                           |
| `start_epoch`             | Starting epoch index (for resuming training)                                                                                                         | Integer value, default: 0                                                                                |
| `num_epochs`              | Number of training epochs                                                                                                                            | Integer value, e.g., 150                                                                                 |
| `eval_epoch_freq`         | Frequency (in epochs) of evaluation during training                                                                                                  | Integer value, default: 1                                                                                |
| `warmup_epoch`            | Number of warm-up epochs where only the posterior (enc-dec) is trained                                                                               | Integer value, default: 1                                                                                |
| `patch_size`              | Patch size for prior keypoint proposals / glimpses                                                                                                   | Integer value, e.g., 16                                                                                  |
| `n_kp_per_patch`          | Number of keypoints to extract from each patch                                                                                                       | Integer value, typically 1                                                                               |
| `n_kp_enc`                | Number of posterior keypoints to be learned                                                                                                          | Integer value, e.g., 45                                                                                  |
| `n_kp_prior`              | Number of prior keypoints / patches                                                                                                                  | Integer value, e.g., 64                                                                                  |
| `topk`                    | Top-k value for plotting keypoints (used for visualization only)                                                                                     | Integer value, default: 10                                                                               |
| `anchor_s`                | Glimpse size as a ratio of image size (posterior patch size)                                                                                         | Floating-point value, e.g., 0.25 for image_size 128                                                      |
| `scale_std`               | Prior standard deviation for object scales                                                                                                           | Floating-point value, typical range [0.15, 1.0] (0.15 for `anchor_s`=0.125, and 0.3 for `anchor_s`=0.25) |
| `offset_std`              | Prior standard deviation for object offsets                                                                                                          | Floating-point value, typical range [0.1, 1.0] (0.1 for `anchor_s`=0.125, and 0.2 for `anchor_s`=0.25)   |
| `obj_on_alpha`            | Prior alpha (Beta distribution) for object-on (transparency)                                                                                         | Floating-point value, e.g., 0.01                                                                         |
| `obj_on_beta`             | Prior beta (Beta distribution) for object-on (transparency)                                                                                          | Floating-point value, e.g., 0.01                                                                         |
| `recon_loss_type`         | Type of reconstruction loss                                                                                                                          | "vgg", "mse"                                                                                             |
| `normalize_rgb`           | Whether to normalize input RGB values tp [-1, 1]                                                                                                     | `true`, `false` (default is `false`)                                                                     |
| `beta_kl`                 | Weight for KL divergence loss on latents                                                                                                             | Floating-point value, e.g., 0.08                                                                         |
| `beta_rec`                | Weight for the reconstruction loss                                                                                                                   | Floating-point value, recommended: 1.0                                                                   |
| `beta_obj`                | Weight for the transparency regularization loss                                                                                                      | Floating-point value, e.g., 0.08                                                                         |
| `kl_balance`              | Balance between attributes and visual features in KL objective                                                                                       | Floating-point value, e.g., 0.01                                                                         |
| `image_size`              | Size (height/width) of the input image                                                                                                               | Integer value, e.g., 128                                                                                 |
| `ch`                      | Number of channels in the input image                                                                                                                | Integer value, typically 3                                                                               |
| `n_views`                 | Number of input camera views                                                                                                                         | Integer value, typically 1                                                                               |
| `features_dist`           | Distribution family for object latent features                                                                                                       | "gauss", other variants as implemented                                                                   |
| `learned_feature_dim`     | Dimension of latent visual features extracted from foreground glimpses                                                                               | Integer value, e.g., 4                                                                                   |
| `learned_bg_feature_dim`  | Dimension of latent visual features extracted from the background                                                                                    | Integer value, e.g., 4                                                                                   |
| `n_fg_categories`         | Number of discrete foreground category embeddings                                                                                                    | Integer value, e.g., 8                                                                                   |
| `n_fg_classes`            | Number of learnable foreground object classes (for clustering/analysis)                                                                              | Integer value, e.g., 4                                                                                   |
| `n_bg_categories`         | Number of discrete background category embeddings                                                                                                    | Integer value, e.g., 4                                                                                   |
| `n_bg_classes`            | Number of learnable background classes                                                                                                               | Integer value, e.g., 4                                                                                   |
| `context_dist`            | Distribution family for context variables                                                                                                            | "gauss", other variants as implemented                                                                   |
| `context_dim`             | Dimension of context latent vector (latent actions)                                                                                                  | Integer value, e.g., 7                                                                                   |
| `n_ctx_categories`        | Number of discrete context category embeddings                                                                                                       | Integer value, e.g., 8                                                                                   |
| `n_ctx_classes`           | Number of learnable context classes                                                                                                                  | Integer value, e.g., 4                                                                                   |
| `ctx_pool_mode`           | Pooling mode for aggregating context information                                                                                                     | "none", "mean", "max" (we used "none")                                                                   |
| `action_condition`        | Flag to condition the model on action inputs                                                                                                         | `true`, `false`                                                                                          |
| `action_dim`              | Dimension of action vector (if action_condition is `true`)                                                                                           | Integer value, e.g., 7                                                                                   |
| `random_action_condition` | Flag to use random action conditioning (for ablations)                                                                                               | `true`, `false`                                                                                          |
| `random_action_dim`       | Dimension of random action vector (if enabled)                                                                                                       | Integer value, e.g., 7                                                                                   |
| `null_action_embed`       | Flag to learn a special embedding for a null/no-op action                                                                                            | `true`, `false` (we used false)                                                                          |
| `image_goal_condition`    | Flag to use image conditioning                                                                                                                       | `true`, `false`                                                                                          |
| `language_condition`      | Flag to use language conditioning                                                                                                                    | `true`, `false`                                                                                          |
| `language_embed_dim`      | Language embedding dimension (e.g. T5-large)                                                                                                         | Integer value, e.g., 1024                                                                                |
| `language_max_len`        | Maximum number of language tokens to allow                                                                                                           | Integer value, e.g., 32                                                                                  |
| `obj_res_from_fc`         | Spatial resolution at which object decoder switches from MLP to convolutional decoding                                                               | Integer value, e.g., 8                                                                                   |
| `obj_ch_mult_prior`       | Channel multipliers for prior CNN of object encoder                                                                                                  | List of integers, e.g., [1, 4, 8]                                                                        |
| `obj_ch_mult`             | Channel multipliers for posterior CNN of object encoder                                                                                              | List of integers, e.g., [1, 4, 8]                                                                        |
| `obj_base_ch`             | Base number of channels for object encoder/decoder                                                                                                   | Integer value, e.g., 32                                                                                  |
| `obj_final_cnn_ch`        | Number of channels in final object CNN layer                                                                                                         | Integer value, e.g., 32                                                                                  |
| `bg_res_from_fc`          | Spatial resolution at which background decoder switches from MLP to convolution                                                                      | Integer value, e.g., 8                                                                                   |
| `bg_ch_mult`              | Channel multipliers for background CNN                                                                                                               | List of integers, e.g., [1, 1, 1, 2, 4]                                                                  |
| `bg_base_ch`              | Base number of channels for background encoder/decoder                                                                                               | Integer value, e.g., 32                                                                                  |
| `bg_final_cnn_ch`         | Number of channels in final background CNN layer                                                                                                     | Integer value, e.g., 32                                                                                  |
| `pad_mode`                | Padding mode for the CNNs                                                                                                                            | "zeros", "replicate" (we used "zeros")                                                                   |
| `num_res_blocks`          | Number of residual blocks in each stage of the CNNs                                                                                                  | Integer value, e.g., 1                                                                                   |
| `cnn_mid_blocks`          | Whether to add residual blocks at intermediate CNN resolutions                                                                                       | `true`, false (we used `false`)                                                                          |
| `use_resblock`            | Whether to use residual blocks in the CNNs at all                                                                                                    | `true`, false (we used `false`)                                                                          |
| `mlp_hidden_dim`          | Hidden dimension of MLP layers used in encoders/decoders                                                                                             | Integer value, e.g., 256                                                                                 |
| `timestep_horizon`        | Number of timesteps for training the dynamics model                                                                                                  | Integer value, e.g., 20                                                                                  |
| `num_static_frames`       | Number of static frames at the beginning of a sequence (burn-in frames)                                                                              | Integer value, e.g., 1                                                                                   |
| `predict_delta`           | Flag to predict deltas between frames instead of absolute coordinates in dynamics                                                                    | `true`, `false` (we used `false`)                                                                        |
| `beta_dyn`                | Weight for KL dynamics loss                                                                                                                          | Floating-point value, e.g., 0.2                                                                          |
| `beta_dyn_rec`            | Weight for dynamics reconstruction loss                                                                                                              | Floating-point value, recommended: 1.0                                                                   |
| `dropout`                 | Dropout rate for transformers                                                                                                                        | Floating-point value in [0.0, 0.5], we used 0.1                                                          |
| `pint_enc_layers`         | Number of transformer layers in particle encoder (PINT encoder)                                                                                      | Integer value, e.g., 1                                                                                   |
| `pint_enc_heads`          | Number of attention heads in particle encoder                                                                                                        | Integer value, e.g., 1                                                                                   |
| `pint_ctx_layers`         | Number of transformer layers used for context modeling                                                                                               | Integer value, e.g., 4                                                                                   |
| `pint_ctx_heads`          | Number of attention heads in context transformer                                                                                                     | Integer value, e.g., 8                                                                                   |
| `pint_dyn_layers`         | Number of transformer layers in dynamics module (PINT dynamics)                                                                                      | Integer value, e.g., 6                                                                                   |
| `pint_dyn_heads`          | Number of attention heads in dynamics module                                                                                                         | Integer value, e.g., 8                                                                                   |
| `pint_dim`                | Hidden dimension of the PINT transformer modules                                                                                                     | Integer value, e.g., 512                                                                                 |
| `run_prefix`              | Prefix/suffix string to append to run directory name                                                                                                 | String, e.g., "_bkl0.08_bdyn0.2_bbal0.01_bobj0.08"                                                       |
| `animation_horizon`       | Number of frames to animate into the future at inference time                                                                                        | Integer value, e.g., 60                                                                                  |
| `cond_steps`              | Number of conditioning steps at inference for the dynamics model                                                                                     | Integer value, e.g., 10                                                                                  |
| `eval_im_metrics`         | Flag to enable evaluation of image-space metrics                                                                                                     | `true`, `false`                                                                                          |
| `ctx_for_eval`            | Flag to use context latents during evaluation (the GT latent actions extracted from the GT video, this is the middle panel in the animations videos) | `true`, `false`                                                                                          |
| `animation_fps`           | Frames per second when saving generated animations                                                                                                   | Floating-point value, e.g., 12.0                                                                         |
| `iou_thresh`              | IoU threshold for object bounding boxes (only for plotting/evaluation)                                                                               | Floating-point value, default: 0.2                                                                       |
